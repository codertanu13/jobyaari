JobYaari is a responsive job portal website built to help users explore the latest government job notifications, 
admit cards, exam results, and career updates all in one place. It’s designed with clear categorization and real‑world
job search usability in mind, making it great both as a learning project and a useful utility for job seekers.

🚀 Live Demo

➡️ https://codertanu13.github.io/jobyaari/

🧠 Overview

JobYaari is a front‑end project that showcases my skills in building a structured, interactive, and user‑friendly website 
that organizes large amounts of information like jobs, exam results, government notifications, and category filters.
The project focuses on clean navigation, accessibility, and real‑world UI patterns often found in job portals.

Key highlights:

Homepage with job categories and filters allowing users to view government job listings.

Search and category navigation tailored to different education levels and sectors (10th pass, graduate, postgraduate).

Latest job openings, exam results, admit card notifications, and quick links easily accessible.

Responsive layout and menu suitable for desktop and mobile viewing.

Social and sharing options plus external links like WhatsApp/Telegram groups for real‑time updates.

📌 Features

✔ Display of 10,000+ latest job notifications and career alerts
✔ Categorized job listing (By education level & sector)
✔ Quick links to admit cards, results, and exam updates
✔ Navigation menu with dropdowns for jobs & tools
✔ External integration for WhatsApp/Telegram group joining
✔ Clean and modern UI, deployed with GitHub Pages

📌 Usage

You can browse through different job categories, search for government jobs, check admit cards and results, and find 
specific job notifications by state or sector. This demo site acts as a practical resource for students, job seekers,
and exam aspirants.

📎 Deployment

This site is deployed using GitHub Pages and automatically reflects updates when code changes are pushed to the main branch.
